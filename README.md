## Overview

This project explores the use of machine learning models to improve insurance risk prediction. It was completed as undergraduate research at the University of Northern Iowa and presented at the INSPIRE Student Research and Engagement Conference.

## Objectives

- Predict insurance risk using machine learning
- Compare multiple predictive models
- Evaluate model performance using standard classification metrics
- Identify important charcteristics influencing insurance purchase decisions

## Tools

- R
- Random Forest
- Probit Regression
- Logistic Regression
- XGBoost
- Data Visualization

## Results

The project compares several machine learning algorithms using:

- ROC Curve
- AUC
- Accuracy
- Sensitivity
- Specificity
- Confusion Matrix
